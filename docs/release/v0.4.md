# Devflow v0.4 Codemap Foundation

## Release Status

v0.4 adds a deterministic Go-first codemap foundation.

## What v0.4 Contains

- `devflow codemap index`
- `devflow codemap search <query>`
- Repo-level `.devflow/codemap/index.json` and `summary.md`
- Demand-level `codemap.md`
- Plan evaluation check `plan.codemap_reference`
- Release-readiness codemap smoke

## Known Limits

- Go-only in v0.4.
- No embeddings.
- No deep call graph.
- No automatic code edits.
- No wiki write-back.

## Verification

- `go test ./... -count=1 -timeout 5m`
- `go vet ./...`
- `go build ./cmd/devflow`
- `git diff --check`
- `scripts\release-readiness.ps1 -Version 0.4.0`
