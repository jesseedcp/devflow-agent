- `plan-context.md` demand artifact.
- `change-scope.md` demand artifact.
- `devflow plan-context refresh`.
- `devflow scope declare`.

- Release-readiness plan-grounding smoke.

## Known Limits

- Scope checks are deterministic and file-based.
- No embeddings.
- No deep dependency graph.
- No automatic code editing.
- No wiki write-back.
