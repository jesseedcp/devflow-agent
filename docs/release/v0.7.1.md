# Devflow v0.7.1

## Summary

v0.7.1 hardens the local wiki knowledge workflow introduced in v0.7.0.

## Changes

- Wiki promote/reject commands now fail with explicit required flag guidance.
- Unsafe wiki entry names now explain the allowed slug characters.
- No storage layout changes and no external wiki providers.

## Verification

- go test ./... -count=1 -timeout 5m
- go vet ./...
- go build ./cmd/devflow
- git diff --check
- scripts\release-readiness.ps1 -Version 0.7.1

## Known Limits

- Wiki distillation remains deterministic text extraction.
- Closeout wiki checks remain warning-level.
