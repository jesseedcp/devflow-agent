# Devflow v1.7.0

v1.7 makes the platform Web UI honest against the real backend.

## Shipped

- Real demand creation endpoint for platform workspaces.
- Rich demand detail response based on `demandflow.InspectWorkspace`.
- HTTP endpoints for supported manual confirmation and acceptance evidence recording.
- React API client methods for workspace creation, demand creation, confirmation, and evidence.
- Page buttons either perform real actions or show a disabled reason when a backend route is intentionally out of scope.
- Page-driven lifecycle test case and API smoke script.

## Boundaries

- LLM runtime stage execution is still CLI/Agent-owned.
- Real rollback execution is not implemented in this release.
- External wiki provider actions are not implemented in this release.
- Browser click automation is not added; the page dogfood is documented as a human test case.
