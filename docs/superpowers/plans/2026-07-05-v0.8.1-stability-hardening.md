# v0.8.1 Stability Hardening Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship v0.8.1 as a small hardening release that reduces test flake risk, completes metrics visibility gaps, and clarifies partial historical metrics.

**Architecture:** Keep v0.8 behavior intact. Make tests less timing-sensitive, surface `metrics.md` consistently as a first-class artifact, render metrics in both workbench snapshot and interactive detail view, and add explicit partial-data caveats to metrics reports. This is a patch release: no new workflow state, no new external adapter, no telemetry, and no post-release observation feature yet.

**Tech Stack:** Go standard library, existing `internal/artifacts`, `internal/runtime/hooks`, `internal/metrics`, `internal/demandflow`, `internal/cli`, PowerShell `scripts/release-readiness.ps1`. No new dependencies.

```go
func TestHookAsyncIsNonBlocking(t *testing.T) {
	eng := NewEngine()
	release := make(chan struct{})
	started := make(chan struct{}, 1)
	eng.AgentRunner = func(prompt string, ctx HookContext) (string, error) {
		started <- struct{}{}
		<-release
		return "done", nil
	}
	defer close(release)

	eng.LoadHooks([]Hook{{
		ID:    "slow-agent",
		Event: EventTurnEnd,
		Async: true,
		Action: Action{
			Type:    ActionAgent,
			Message: "inspect later",
		},
	}})

	start := time.Now()
	res := eng.RunHooks(HookContext{EventName: EventTurnEnd})
	elapsed := time.Since(start)
	if elapsed > 250*time.Millisecond {
		t.Errorf("async hook blocked the caller for %v", elapsed)
	}
	if len(res) != 1 || res[0].Output != "(async)" {
		t.Errorf("expected async stub result, got %#v", res)
	}
	select {
	case <-started:
	case <-time.After(2 * time.Second):
		t.Fatal("async agent hook did not start")
	}
}
```

This keeps the behavior under test but removes process startup, shell scheduling, and `sleep` from the nonblocking assertion.

Add helper:

```go
func demandCaveats(out DemandMetrics, events []artifacts.Event) []string {
	if len(events) == 0 {
		return []string{"no events recorded"}
	}
	var caveats []string
	if out.VerificationRuns == 0 {
		caveats = append(caveats, "no verification events")
	}
	if out.AcceptancePasses+out.AcceptanceFailures+out.AcceptanceBlocked == 0 {
		caveats = append(caveats, "no acceptance evidence events")
	}
	if out.WikiCandidatesDistilled == 0 && out.WikiPromoted == 0 && out.WikiRejected == 0 {
		caveats = append(caveats, "no wiki decision events")
	}
	return caveats
}
```

	out.PartialDemandCount++
